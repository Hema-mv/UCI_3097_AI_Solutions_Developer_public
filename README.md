## Naming Convention (Professional Standard)

To keep the repository consistent and easy to search, content files follow these formats:

- `lesson_<module>.<section>_<topic>.md`
- `pacesetter_<module>.<section>_<topic>.md`
- `poc_<module>.<section>.<item>_<topic>.md`
- `glab_<module>.<section>[.<item>]_<topic>.md`
- `all_links.md`

Examples:

- `lesson_933.1_understanding_artificial_intelligence.md`
- `pacesetter_938.4_prompt_engineering_with_openai_api.md`
- `poc_939.3.1_vector_databases_with_pinecone.md`
- `glab_938.8_gpt4all_local_installation_and_usage_guide.md`
